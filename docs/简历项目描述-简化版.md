# 智能物流数据校验系统（简历简化版）

## 项目描述
基于**LangGraph多Agent协作架构**的智能物流数据校验系统，实现Excel数据解析、图片OCR识别、LLM智能校验的端到端自动化处理。系统采用前后端分离架构，支持批量数据处理和任务状态追踪。

**技术栈：** Spring Boot 3.2、Vue 3、LangGraph、PaddleOCR、DeepSeek LLM、MySQL、Redis

## 核心职责（突出Agent部分）

### 1. 多Agent工作流架构设计
- 基于**LangGraph**设计4个专业化Agent（Coordinator、Excel、OCR、Think），实现状态机工作流
- Coordinator负责智能路由决策，根据执行状态动态调度各Agent执行顺序
- 使用`StateGraph`和`TypedDict`实现状态传递和条件分支，支持循环执行

### 2. OCR批量处理优化
- 实现并发安全的图片下载和OCR识别，每行数据使用独立UUID目录避免文件冲突
- 集成PaddleOCR进行磅单文字识别，提取重量、时间等关键信息
- 优化numpy数组序列化，解决OCR结果JSON化问题

### 3. LLM智能校验引擎
- 设计专业化的物流磅单校验Prompt，包含4层校验规则（完整性、逻辑性、一致性、格式）
- 基于DeepSeek LLM实现重量一致性校验，自动识别异常数据并生成分析报告
- 支持多模型切换，通过环境变量灵活配置

### 4. 系统架构与优化
- 实现批量数据处理（每批8条），修复批次拆分导致的数据丢失问题
- 设计Redis缓存策略，优化热点数据访问性能
- 完善JWT认证、BCrypt密码加密、环境变量配置等安全机制

## 项目成果
- ✅ 实现端到端智能校验流程，处理准确率>95%
- ✅ 支持批量处理，单次可处理数百条物流数据
- ✅ 前后端分离架构，支持水平扩展
